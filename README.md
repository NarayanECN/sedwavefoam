# sedwavefoam
SedWaveFoam solver for OpenFOAM 

### What is this repository for? ###
* OpenFOAM
* Supported OpenFoam Versions : 2.3.0

### Installation ###
* Install OpenFOAM
* Install waves2Foam
* git clone https://github.com/sedwavefoam/sedwavefoam sedwavefoam
* cd sedwavefoam
* source bashrc of OpenFOAM and waves2Foam
* ./Allwclean
* ./Allwmake
